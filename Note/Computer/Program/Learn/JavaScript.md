
## React


##### 创建React项目

###### 使用`Create React App`
```bash
npx create-react-app my-app
cd my-app
npm start
```






###### 使用`Vite`
```bash
npm create vite@latest my-app -- --template react
cd my-app
npm install
npm run dev
```






项目结构
```bash
my-app/
├── node_modules/   # 依赖库
├── public/         # 静态资源（HTML、图片等）
├── src/            # 源代码
│   ├── App.jsx     # 主组件
│   ├── main.jsx    # 入口文件
│   └── ...
├── package.json    # 项目配置和依赖
├── vite.config.js  # Vite 配置（如使用 Vite）
└── ...
```
